# FS_DAQ_System

This is repo used for developing a DAQ system for University of Leed's formula student conbustion engine car

## Project Overview

The DAQ system is made of 5 seperate parts - 

- Sensors
- Teensy 4.1
- Nano 2040 Connect
- Display
- Server
