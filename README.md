# FS_DAQ_System

This is a repo used for developing a DAQ system for the University of Leeds' Formula Student combustion engine car

## Project Overview

The DAQ system is made of 5 seperate parts - 

- Sensors
- Teensy 4.1
- Display
- RF Module
